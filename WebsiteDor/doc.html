Our Project

In this project we try to reduce the waiting time in a junction with traffic light using a simple IOT device.

To achive that goal we have two main parts in our solution:

The first one is collecting the data to process.

The second is processing the data and deciding how to manage the traffic.


Collecting the data

To collect the datawe use a camera that shoots the junctionand sends a snapshot every second. We process the snapshots using machine learning algorithm and compute the number of cars coming into every lane.


Process the data

Given the number of cars coming from every lane we created an algorithm that decide which light will be green in the next period.
